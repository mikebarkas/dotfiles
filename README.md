## Dotfiles

Home directory configuration files.

* Setup Links _(see readme in setup)_
* Tmux
* Vim
* Mutt (not finished)
* Zsh
* Git
* Npm
* Hg

#### Setup

Run the setup script, see setup readme for details.<br/>
`setup/./init.sh`

Symlinks are created to the configuration files in the _.dotfiles_ directory.
